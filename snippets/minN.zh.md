### 明尼苏达州

返回`ñ`来自提供的数组的最小元素. `如果`ñ

大于或等于提供的数组长度,则返回原始数组(按升序排序). `使用`中的Array.sort()`结合传播算子(`...`)创建数组的浅层克隆,并按升序对其进行排序`array.slice()`获取指定数量的元素. 试试第二个参数,`ñ,得到一个元素的数组. 

```js
const minN = (arr, n = 1) => [...arr].sort((a, b) => a - b).slice(0, n);
```

```js
minN([1, 2, 3]); // [1]
minN([1, 2, 3], 2); // [1,2]
```
